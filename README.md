# Bitly_ — URL Shortener App (v2)

An updated Angular application that integrates the **Bitly API** to shorten, track, and manage URLs.

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | Angular, TypeScript |
| API | Bitly REST API |
| Styling | Bootstrap, CSS3 |

## Features

- Shorten long URLs via Bitly API
- Copy shortened link to clipboard
- Link history and management
- Clean, minimal UI
